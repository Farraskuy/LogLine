# LogLine

Offline-first Flutter logbook and notes app boilerplate.

This repository currently contains a minimal Flutter app shell and ready-to-use service scaffolding for:

- Hive local storage
- MongoDB Atlas Data API client
- Offline sync queue
- Connectivity monitoring
- Local secure session and biometric auth
- Markdown helpers
- Camera text extraction orchestration
- YOLO/TFLite text-region detector wrapper
- ML Kit OCR service

The UI is intentionally empty so feature screens can be implemented from the Figma design later.
